Hexo+GitHub搭建个人博客

发表于 2020-03-21 分类于 Hexo 阅读次数：

Ready 准备工作

在开始前，你需要准备以下工作

  • github账号
  • nodejs、npm环境，了解基础知识
  • git客户端
  • 个人域名（可选）

创建 Github 仓库

使用 {username}.github.io 作为仓库名称，初始化时带上README创建

创建后在仓库setting看到可以通过域名访问

Hexo安装、配置

Hexo 版本 : 4.2.0，根据官方教程安装、运行，即可看到效果

Hexo上传到Github

已提前对git进行配置，如第一次使用也是上网搜索一下

对Hexo的_config.yml配置文件进行修改

先后执行hexo clean hexo g hexo d命令（具体命令意思查看官方文档），提交到github后，再访问域名即可看到效果

域名绑定（可选）

通过ping命令，获取github域名对应的ip地址

再把自己域名解析此IP（这里我使用的是阿里云域名）

在仓库根目录中创建CNAME文件，把需要绑定的域名填入（注意填入的域名需要）

最后访问绑定的域名即可看到效果…
